# 🗂️ Portfolio Main Page — Plantilla

Una plantilla minimalista y lista para usar para crear tu portfolio personal. Incluye navbar, animación de texto y estilos base con una estética oscura y moderna.

## ✨ Características

- Navbar responsive con logo y links de navegación
- Animación de escritura (typewriter) en el nombre
- Fondo oscuro con tipografía Lexend de Google Fonts
- Iconos de Font Awesome integrados
- Estructura limpia y fácil de personalizar

## 🚀 Cómo usarla

1. Clona el repositorio:
   ```bash
   git clone https://github.com/abelochooo/portfolioMainPage.git
   ```
2. Abre `index.html` en tu navegador o con Live Server en VS Code.
3. Edita `index.html` y `style.css` con tu nombre, links y colores.

## 🛠️ Personalización

| Qué cambiar | Dónde |
|---|---|
| Tu nombre | `index.html` → `.portfolio-h1` |
| Links del navbar | `index.html` → `.nav-bar a` |
| Tu logo | `images/` → reemplaza el SVG |
| Colores y fuentes | `style.css` |

## 📁 Estructura
